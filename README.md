# Capstone Project – Fashionista (E-Commerce Website)

This is my Capstone Project for Web Development Lab.  
The project is a fully responsive **E-Commerce Fashion Store UI** made using **HTML5 + CSS3 only**.

The website includes a hero banner, flash sale bar, featured product grid, testimonials, login form, newsletter subscription, and premium membership section.  
All images used are stored in the **images/** folder.

---

## Features Implemented

### Fully Responsive (Mobile → Tablet → Desktop)
The layout adjusts smoothly across breakpoints:
- **≤480px** – Mobile view  
- **≤768px** – Tablet view  
- **≥1024px** – Desktop view  

### Modern CSS Techniques Used
- CSS Flexbox  
- CSS Grid (Product cards layout)  
- CSS Variables  
- Hover transitions  
- Box-shadows & border-radius  
- Keyframe Animation (Hero title floating effect)  
- Responsive image scaling  
- Focus styles for accessibility    

### Semantic HTML
Used tags:  
`header`, `nav`, `main`, `section`, `article`, `footer`, `form`, `label`, `button`, `img`

### Accessibility Features
- Alt text on all images  
- Screen-reader-only labels using `.visually-hidden`  
- ARIA roles: navigation, hero section, testimonials, forms  
- High contrast buttons & focus outlines
